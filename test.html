Data narratives for result: highly Connected Drugs

Data Narrative 1: Execution view     

The Abstract Global Workflow2 method was run on the model info updated, Mtb Structures tar gz, homology models list txt, tb protein info, solved tb structures list txt, drug key, smap properties, template pdbs, drug binding sites list txt datasets, whith parameter PVALUE0.00001 set to 0.00001, The highly Connected Drugs results are stored online (DOI https://dx.doi.org/10.6084/m9.figshare.776887.v1).

Data Narrative 2: Data view     

The highly Connected Drugs results have been derived from the model info updated, drug binding sites list txt, homology models list txt, smap properties, template pdbs, tb protein info, solved tb structures list txt, and drug key datasets.

Data Narrative 3: Method view     

The method Abstract Global Workflow2 performs 4 main types of analysis on the input datasets. First, the Compare Ligand Binding Sites V2 step processes the datasets, then the data is analyzed by the Compare Ligand Binding Sites V21, Compare Dissimilar Protein Structures steps and finally, Docking produces the end results.

The highly Connected Drugs results are the product of the Make Graph Network, a Data Visualization step.
