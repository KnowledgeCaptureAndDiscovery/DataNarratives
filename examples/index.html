Data Narratives: Examples

A list of Data Narratives for workflows of different domains

Here we list some of the Data Narratives examples we have been building automatically from different workflows on a web analytics domain:

  1. Narratives of an image resulting of a topic detection workflow.
  2. Narratives for the image accessibility analytics depicting the accesses of users to a website.
  3. Narratives for the result on understanding the user access workflow.
  4. Narratives for the results of a workflow for trying to identify the problematic users of a website.
  5. Narratives for the visualization of the trending topic of a website.

Examples from other domains will be added soon
