Data Narratives: Examples

A list of Data Narratives for workflows of different domains

Here we list some of the Data Narratives examples we have been building automatically from different workflows:

  1. Topic modelling: description of how a term-topic matrix visualization was obtained.
  2. Genomics: different views of how a set of highly connected drug results were obtained from the TB-drugome workflow. (Under construction)
  3. Water quality: narratives from a metabolism result dataset, obtained after running a raeration empirical method.
